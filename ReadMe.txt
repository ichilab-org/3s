「Personal」フォルダについて
各自のプログラムファイルは、Personalフォルダ以下に作成すること。
Personalフォルダ以下に、自分の学籍番号と苗字を使ったフォルダを作ること。
例　br180001_ichikawa （全部小文字、アンダースコアで連結）

## S4のWorkspaceについて
`Personal`フォルダ以下の各自のフォルダに`Workspace`をワークスペースに設定すること

```
Personal
|- bp18000_shibaura
    |- Workspace
```
